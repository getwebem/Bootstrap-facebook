#Social Network app
###**Getting started**
###Technologies Used
####HTML5, CSS3, Bootstrap 3.3.7, Photoshop
 - Run :  `git clone https://github.com/getwebem/facebook-clone.git`
 - Run :  `cd facebook-clone`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
